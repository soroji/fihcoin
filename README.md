# FIH Ecosystem (FIH & FIHPT)

https://www.fin.money

## Overview
FIH Ecosystem consists of two tokens designed to work together:

- **FIH**: A meme token deployed on the Solana network
- **FIHPT**: A utility token deployed on opBNB, powering the FIH ecosystem

Together, these tokens form a hybrid ecosystem combining community-driven value and platform-based utility rewards.
